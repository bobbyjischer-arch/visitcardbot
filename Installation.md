# 📱 Установка O.R.I.O.N. на iPhone через Windows

---

## Что тебе нужно
- Windows ПК
- iPhone с кабелем
- Apple ID (обычный, бесплатный)
- Аккаунт на GitHub (бесплатный)

---

## Шаг 1 — Загрузи код на GitHub

1. Зайди на **github.com** → создай новый репозиторий, назови `orion-ios`
2. Загрузи все файлы из этой папки в репозиторий
3. После загрузки GitHub **автоматически** запустит сборку

---

## Шаг 2 — Скачай собранный IPA

1. Открой репозиторий на GitHub
2. Нажми вкладку **Actions** (сверху)
3. Найди последний запуск **"Build O.R.I.O.N. IPA"** — дождись зелёной галочки ✅
4. Нажми на него → прокрути вниз → раздел **Artifacts**
5. Скачай **ORION-IPA.zip** → распакуй → внутри будет `ORION.ipa`

---

## Шаг 3 — Установи Sideloadly на Windows

1. Скачай с официального сайта: **https://sideloadly.io**
2. Установи и запусти

---

## Шаг 4 — Установи приложение на iPhone

1. Подключи iPhone к ПК кабелем
2. На iPhone нажми **«Доверять»** если появится запрос
3. В Sideloadly:
   - Перетащи `ORION.ipa` в окно программы
   - В поле **Apple Account** введи свой Apple ID
   - Нажми **Start**
   - Введи пароль Apple ID когда попросит
4. Дождись надписи **"Done"**

---

## Шаг 5 — Доверяй разработчику на iPhone

⚠️ Без этого шага приложение не откроется!

1. Открой **Настройки** → **Основные** → **VPN и управление устройством**
2. Найди свой Apple ID в разделе **«Приложения разработчика»**
3. Нажми **«Доверять [твой Apple ID]»** → подтверди

---

## Шаг 6 — Первый запуск

1. Открой **O.R.I.O.N.** на iPhone
2. Разреши геолокацию → выбери **«Всегда»**
3. Перейди в **Настройки** → введи IP своего сервера
4. Нажми **«Запустить слежение»**

---

## ⚠️ Важно: переустановка раз в 7 дней

Apple ограничивает бесплатные аккаунты — сертификат живёт **7 дней**.

**Решения:**
- Переустанавливай через Sideloadly раз в неделю (2 минуты)
- Купи Apple Developer аккаунт за $99/год → будет работать год
- Используй AltStore (альтернатива Sideloadly, тоже 7 дней)

---

## 🔧 Настройка сервера (IP адрес)

Чтобы iPhone мог слать данные на твой сервер:

1. На компьютере с сервером узнай локальный IP:
   - Windows: `ipconfig` в командной строке → IPv4 Address
   - Обычно выглядит как `192.168.1.XXX`

2. В приложении O.R.I.O.N. → Настройки → URL сервера:
   ```
   http://192.168.1.XXX:8000
   ```

3. iPhone и компьютер должны быть в **одной Wi-Fi сети**

> Для работы вне дома нужен внешний IP или VPN — это Lvl 5 задача.

---

## Если что-то пошло не так

| Проблема | Решение |
|----------|---------|
| Sideloadly просит пароль снова | Введи пароль Apple ID — это нормально |
| "Untrusted Developer" | Шаг 5 — доверяй разработчику в настройках |
| Сборка упала на GitHub | Actions → посмотри лог ошибки → напиши мне |
| Сервер недоступен | Проверь IP, оба устройства в одной сети? |
| Геолокация не работает в фоне | Настройки → ORION → Геолокация → «Всегда» |
